# Pagination

Node.js aplication with fake products from MongoDB using fake.js and Bootstrap pagination.

## Installation

You need to have installed Node.js and MongoDB. You can download and install it from official websites:

[Node.js](https://nodejs.org/en/)

[MongoDB](https://www.mongodb.com/download-center?jmp=nav#community)

and install necessary dependencies to work:
```
npm install body-parser ejs express faker mongoose --save
```

## Usage

run server.js from terminal 
```
node server.js
```

on the browser -> http://localhost:8080/
